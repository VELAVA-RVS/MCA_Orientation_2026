# MCA Orientation 2026 Website

## Project Description
This project is a multi-page website developed using HTML and CSS for the MCA Orientation 2026 programme. Each collaborator is responsible for creating and maintaining a specific webpage.

## Technologies Used
- HTML5
- CSS3

## Folder Structure

```
MCA_Orientation_2026/
│
├── index.html
├── about.html
├── schedule.html
├── speakers.html
├── gallery.html
├── contact.html
├── style.css
├── images/
└── README.md
```

## How to Run
1. Download or clone the project.
2. Open `index.html` in a web browser.
3. Navigate through the website using the menu.
